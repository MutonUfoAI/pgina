User Account Control

The User Account Control is handled as you know it from Windows

You can use local or remote accounts with pGina.

While your request is beeing processed you’ll get a message box. The whole pGina plugin chain is called as long as this user is a pGina user, there is no diffence to a normal login.

If there is an error you’ll get another message box telling you what went wrong.
In the case of this screenshot someone elevated a program by using runas. If another user is trying to elevate a program using the same administratively account (if a pGina user), than you’ll get this error. Local or Domain accounts won’t run in such a problem, because only pGina users are monitored.
